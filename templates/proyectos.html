Constructor

The Constructor is the core web application where young creators build their own video games. It features multiple interactive views, including the Constructor, Gameplay, Node Editor, Card Editor, and even a Python Code Editor.
Using computer vision, the system can recognize physical cards, while AI assists in explaining what each card, node, or command does, helping learners understand programming concepts as they create.
Each chapter concludes with the option to export or publish their projects, allowing students to share their creations with friends, classmates, or family.
This section invites you to explore how the Constructor works, a playful and intelligent environment designed to make learning game development feel like pure magic.
